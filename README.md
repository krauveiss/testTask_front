# testTask_front

SPA для ТЗ "Книга рецептов" на `Vite + React + TypeScript`.

## Что внутри

- CRUD по продуктам и блюдам через Laravel API
- фильтрация, поиск и сортировка продуктов
- фильтрация и поиск блюд
- автопересчёт КБЖУ блюда по составу
- поддержка макросов категорий в названии блюда
- блокировка недоступных флагов блюда по составу
- обработка `409` при удалении продукта, который уже используется в блюдах
- базовые unit-тесты для правил расчёта

## Запуск

1. Установить зависимости:

```bash
npm install
```

2. Создать `.env` в корне фронта:

```env
VITE_API_URL=http://127.0.0.1:8000/api
```

3. Запустить фронт:

```bash
npm run dev
```

4. Для сборки:

```bash
npm run build
```

5. Для тестов:

```bash
npm run test
```

## Важно по фото

Бэк принимает `photos` как массив ссылок и/или загружаемых файлов. Для локально загруженных файлов Laravel должен быть настроен так, чтобы сохранённые пути реально открывались из браузера. Если нужны публичные URL, проверь `APP_URL`, файловый диск и режим отдачи файлов на стороне бэка.
